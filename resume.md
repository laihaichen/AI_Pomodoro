# 项目介绍 — 简历用

## 版本 A：适合投递给游戏公司

**智能番茄钟 × 文字 RPG 系统** — 全栈独立项目

设计并独立开发了一套将番茄钟时间管理与文字 RPG 深度耦合的桌面应用系统。玩家的每次学习记录会触发一次命运判定（基于概率、健康度、超时惩罚等多维状态机），驱动模拟人生式的角色叙事。系统包含完整的游戏设计：6 级命运值区间映射事件体系、幸运系统（干预卡/宿命卡双卡储蓄机制）、阶段性里程碑任务、角色死亡与继承切换（Survivor Mode）、以及基于 Gemini API 的 AI NPC 伙伴系统——支持多轮上下文对话，角色性格由 Markdown 档案驱动，实时注入游戏状态面板作为 prompt 上下文。技术栈覆盖 Python/Flask 后端、原生 JS 实时仪表盘、SQLite 双写一致性、macOS AppleScript 跨应用自动化（Alfred + Chrome），以及 800+ 行游戏规则文档作为 AI system prompt 的工程化管理。

## 版本 B：适合投递给非游戏企业

**智能番茄钟学习管理系统** — 全栈独立项目

独立设计并开发了一套基于 Flask 的实时学习追踪系统，通过低心智负担的轻量化激励机制解决长时间学习场景下的拖延与疲劳问题。系统核心是一个多维状态引擎：实时计算时间区间、健康度衰减、偏移量校验等指标，并通过 WebSocket 级轮询将 20+ 项状态同步至前端仪表盘。集成 Google Gemini API 实现 AI 对话助手功能，支持多轮上下文管理（含 token 预算控制与历史截断策略）、角色化 system prompt 工程、以及游戏状态的实时上下文注入。基础设施层实现了 SQLite + JSON 的双写一致性保障、macOS 原生 AppleScript 跨应用自动化编排（浏览器操控 + 桌面工作流触发）、基于文件 IPC 的异步 Agent 协作模式、以及模块化的条件-效果技能引擎。前端使用原生 JavaScript 构建，无框架依赖，包含实时图表、多步骤 Modal 交互和响应式布局。
